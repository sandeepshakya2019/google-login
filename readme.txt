** Required
node.js
express.js (npm repository)
mongoDB database 

1) First login to the mongoDB cloud and create a cluster and a database user create a username and password put the password in the app.js file
2) now do the npm i (in installed automatically all npm repositories and node_modules)
3) run the app.js using the command node or nodemon app.js (in terminal)
4) open the browser type localhost:3000 to view this 
5) if you want to skip the step 1 then you can run the mongoDB shell on the localhost itself use the command mongod to run the shell and 
use the mongo to run the mongodb shell to see the collections and documents
